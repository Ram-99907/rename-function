📂 Rename File Using C
📌 Description
This project demonstrates how to rename a file using the rename() function in C.
The program takes the old file name and new file name as input from the user and renames the file if it exists.

🛠️ Concepts Used
File Handling in C

rename() function

Basic Input/Output

Error checking

💻 How It Works
User enters the existing file name.

User enters the new file name.

The program calls rename(oldName, newName).

If successful → Displays success message.

If error → Displays error message.

▶️ Sample Output
Enter old file name: old.txt
Enter new file name: new.txt
